HomeNetSec Skill (source)

This repo folder is structured as an OpenClaw skill.

- SKILL.md defines the skill metadata + minimal usage notes.
- scripts/ will contain the pipeline runner and helper scripts.
- references/ will contain documentation used by the agent.
- assets/ will contain templates (e.g., docker-compose.yml) used by scripts.

Generated data (PCAPs, Zeek logs, RITA DB, reports) should not be committed.
